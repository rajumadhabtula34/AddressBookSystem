# UC5 - Multiple Address Books

## Objective

Refactor the Address Book System to support multiple Address Books, where each Address Book has a unique name.

## Features

* Create multiple Address Books.
* Assign a unique name to each Address Book.
* Store Address Books using a `HashMap<String, AddressBook>`.
* Add contacts to different Address Books.
* Display all available Address Books.
* Manage multiple Address Books within a single Address Book System.

## Technologies Used

* Java
* Eclipse IDE
* Collections Framework (`HashMap`, `ArrayList`)
* Object-Oriented Programming (OOP)

## Classes

* `Contact` – Stores contact information.
* `AddressBook` – Manages multiple contacts.
* `AddressBookSystem` – Manages multiple Address Books.
* `AddressBookMain` – Driver class for execution.

## OOP Concepts Used

* Class
* Object
* Association
* Encapsulation
* Collection Framework

## Output

* Create multiple Address Books.
* Add contacts to different Address Books.
* Display Address Book names and their contacts.
